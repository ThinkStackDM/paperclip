# Etsy SEO, Pricing & Photography

Etsy ranks on relevance + a quality score (CTR, conversion, recency) inside a visual grid where the
first image does ~80% of the clicking. Fill the two fields that decide outcome: searched words + the clicked image.

## Titles + tags (13-tag budget)
- Front-load the title with the exact multi-word phrase a buyer types ("Monstera Wall Art Printable").
- Mine demand from Etsy autocomplete + "related searches" chips + top-competitor tags (visible on their listing).
- **All 13 tags, each a multi-word phrase** (single words waste a slot — Etsy matches words inside phrases); no two tags sharing exact words; mix head + long-tail. Use all attributes (color/orientation/room) — free ranking signals.

## Pricing digital wall-art
- Anchor to the benchmark band; price at/just below the top-sellers' median for the query (too cheap = low-quality signal).
- **Sets out-earn singles** — bundle 3–6 sizes/colorways as one listing (higher AOV, same file work).
- Intro sale (badge lifts CTR) rather than permanently underpricing. Log price changes as marketing-ops experiments; never A/B price + photo together.

## Imagery (4 tiles)
1. The click: art shown large + styled in a believable room mockup (decides grid CTR).
2. Flat true-color view of the print (trust).
3. What's included — sizes/ratios as a labeled grid (kills "what do I get?" hesitation).
4. Second styled context for range.
Generate via image-gen-ops; QA each through the etsy-listing-ops visual-truth gate (mobile readability) before publish.

## Deliverable
A complete listing-pack row — title, 13 tags, attributes, price + sale plan, 4-tile brief — for the
etsy-listing-ops publish manifest. Live copy is board-gated; you draft, the board posts.
