# Episode Script Standard

Use this for Stage 1-2 faceless YouTube work where the agent must lock research and draft spoken narration, not just write plausible prose.

## Entry gate

Do not draft until the package claim, verified numbers, worked example, and proof visuals are clear enough to name. If those inputs are missing, say what is missing instead of writing around it.

## Research law

- recompute the worked example from source
- keep each headline number attached to the exact rung or scenario that owns it
- note competitor angle so the episode has a distinct promise
- map each claim to the chart, table, screenshot, calculator, or b-roll beat that proves it

## Script law

- first 20 seconds must carry a number, rule, or mistake
- write for the ear: contractions, one idea per sentence, define terms on first use
- ban meta narration like `in this episode` or `today we're going to`
- title and thumbnail facts must appear in the script and on a proving visual
- every substantive line should name the proving visual
- YMYL disclosure is one sentence max
- for single-narrator channels, each section is one continuous narrator take; do not solve reveals with spliced per-part narration

## Handoff check

Before you finish, confirm:

- the hook lands immediately
- the package claim is proven, not just repeated
- every section reads cleanly out loud
- no unresolved number is hiding inside polished copy
