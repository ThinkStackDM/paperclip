# Agent QA Pack v1

This pack turns a handful of repeatable QA habits into reusable operating
assets for an AI-assisted team.

Included files:

- `Agent-QA-Pack-Guide.md`
- `01-Ship-It-QA-Checklist.md`
- `02-PDF-Report-Contract-Template.md`
- `03-Terminal-Bench-Loop-Pattern.md`
- `04-Generic-QA-Gate-Patterns.md`
- `05-Usage-License.txt`

Quick start:

1. Read `Agent-QA-Pack-Guide.md`.
2. Copy the checklist and report template into your own issue or project flow.
3. Adapt the terminal-bench loop and QA gates to your own stack and approvers.

Honesty note:

This pack was built from a Claude-agent operating style. The checklists and
gates are portable. Any prompts, issue flow, or skill-file snippets should be
adapted to your own tools, naming, and review path.
